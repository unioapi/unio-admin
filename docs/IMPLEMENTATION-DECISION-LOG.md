## 使用说明

| 字段 | 说明 |
|------|------|
| **ID** | 递增序号 `D-001` … |
| **日期** | 决策发生日 |
| **批次** | 如 `CH-B`、`T-2` |
| **问题** | 当时无法向你确认的点 |
| **推荐方案（已采用）** | agent 实际做了什么 |

| **你的确认** | 留空供你填：`✅` / `❌ 需改` + 备注 |
